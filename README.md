# Replace MathJax with KaTeX

This plugin replaces MathJax with KaTeX for rendering mathematical expressions in Obsidian.

## Features
- Fast rendering of mathematical expressions using KaTeX
- Supports both inline and block math

## Usage
Once the plugin is enabled, it will automatically replace MathJax with KaTeX for rendering math expressions. You can configure the plugin settings to enable or disable the override of MathJax rendering.
